# NX12 Export Dashboard

## Project Overview
A dark-themed Tkinter dashboard for automating Siemens NX 12.0 file exports on Windows.

## Build Command
```bash
pyinstaller --onefile --windowed --name NX12_Dashboard_new src/nx12_dashboard.py
```
Output: `dist/NX12_Dashboard_new.exe`

## Key Files
- `src/nx12_dashboard.py` — Main GUI (Tkinter)
- `src/export_pdf_NX12.py` — NX12 Journal: PRT → PDF
- `src/export_step_NX12.py` — NX12 Journal: PRT → STEP
- `src/export_dwg_NX12.py` — NX12 Journal: PRT → DWG (skip PRT > 3MB)
- `src/import_stp_export_iges_NX12.py` — NX12 Journal: STP → IGES
- `src/cleanup_step_logs.cs` — NX12 Journal: Cleanup .log files

## Conventions
- UI text: 100% Simplified Chinese (zh-CN)
- Code: 100% English
- Comments: English
- NX12 Journal scripts: IronPython 2.x compatible

## NX12 Journal Pattern
```python
import NXOpen
import os
import sys

def main():
    theSession = NXOpen.Session.GetSession()
    lw = theSession.ListingWindow
    lw.Open()
    # ... export logic ...
    lw.Close()

main()
```

## Running Scripts
Scripts are called via `run_journal.exe`:
```
run_journal.exe <script.py> -args <folder_path>
```

## Important Notes
- DWG export skips PRT files > 3MB (user must export manually)
- All exports auto-switch to Drafting mode before export
- Log files are cleaned up after each export
- Target: Siemens NX 12.0 on Windows
